Detailed Description

A Dispatcher that uses TmEvent to send messages.

Whenever the Dispatcher executes a Dispatch(), observers will be notified of any event (Key) for which they have registered.

EventDispatcher dispatches TmEvents.

The EventDispatcher is the only Dispatcher currently released with Strange (though by separating EventDispatcher from Dispatcher I'm obviously signalling that I don't think it's the only possible one).

EventDispatcher is both an ITriggerProvider and an ITriggerable.
